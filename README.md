# 🔧 NetworkRepair

> Batch script for **network repair** — NLA probing, BFE/Dnscache/MpsSvc/DHCP/DUSM services, adapter disable/enable cycle.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **Registry** | NetworkConnectivityStatusIndicator, NlaSvc parameters |
| 🔧 **Services** | BFE, Dnscache, MpsSvc, DHCP, DUSM, NlaSvc, WlanSvc, etc. |
| 🔄 **Adapter Reset** | Disables then re-enables adapters (index 0-5) |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 |
| **Privileges** | Administrator |

---

## 🚀 Usage

```cmd
:: Run as Administrator
NetworkRepair.cmd
```
